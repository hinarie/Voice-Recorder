# Simple Voice Recorder

### Возможности

- 🎤 Запись голоса
- 👤 Хранение в самом файле, в формате .wav

### Технологии
- Python

## Структура проекта

```
venv/
.gitignore
image.png
main.py
requirements.txt
```

## Установка

1. Клонировать репозиторий

```bash
git clone https://github.com/hinarie/Voice-Recorder.git
```

2. Создать виртуальное окружение

```bash
python -m venv venv
```

3. Активировать его

Windows

```bash
venv\Scripts\activate
```

4. Установить зависимости

```bash
pip install -r requirements.txt
```

5. Запустить

```bash
python main.py
```
